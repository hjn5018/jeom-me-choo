# jeom-me-choo
**점심메뉴추천 사이트**
매일 매일 점심메뉴를 고민하는 사람들을 위한 게시판 사이트
2024.04.01 - 2024.04.05 (5 days)

- 김환호 : 글 목록, 글 조회 페이지
- 장석천 : 회원가입, 로그인, 프로필 이미지, 메인페이지, 헤더 메뉴 (총괄)
- 한진용 : 게시글 CRUD (팀장)
- 류종현 : 댓글 CRUD, 추천 기능, 푸터
